# CyberShield - Cyber Awareness & Digital Safety Platform

A premium, production-ready landing page for a cyber awareness and digital safety platform designed for students.

## 🎨 Design Philosophy

- **Light Theme**: Clean, trustworthy, government/academic grade design
- **Premium Feel**: Sophisticated animations and micro-interactions
- **Modern & Calm**: No flashy effects, focused on clarity and trust
- **Typography**: DM Sans for a refined, professional look
- **Color Palette**: Soft blues and cyans with high-contrast text

## 🚀 Tech Stack

- **React 18** - Modern React with hooks
- **Vite** - Lightning-fast build tool
- **Lucide React** - Beautiful, consistent icons
- **Pure CSS** - No Tailwind or framework dependencies
- **Vanilla animations** - Smooth, performant CSS animations

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎯 Design Decisions

1. **DM Sans Typography**: Chosen over generic fonts (Inter, Roboto) for a distinctive, premium feel while maintaining excellent readability.

2. **Light Theme**: Selected for trustworthiness and academic credibility. Perfect for educational platforms.

3. **Gradient Accents**: Blue-to-cyan gradients used sparingly for CTAs and highlights, never overwhelming.

4. **Micro-interactions**: Every hover state, button press, and scroll reveal is carefully crafted for delight without distraction.

5. **Animated Illustration**: Custom-built orbiting security icons create visual interest while reinforcing the cyber safety theme.

6. **Performance First**: Pure CSS animations, lazy loading patterns, and optimized re-renders.

## 🔧 Customization

### Colors
Edit the CSS variables or gradient colors in `CyberSafetyLanding.css`:
- Primary Blue: `#2563eb`
- Cyan Accent: `#06b6d4`
- Background: `#f8fafc`, `#eff6ff`
- Text: `#0f172a`, `#64748b`

### Typography
Change the font by updating the Google Fonts import in the CSS file.

### Content
All text content is in `CyberSafetyLanding.jsx` for easy editing.
